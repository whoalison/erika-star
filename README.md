Erika Star é um jogo desenvolvimento como projeto da disciplina de Modelagem e simulação de sistemas computacionais. O jogo acompanha a jornada de Erika sobre o conhecimento de Diagramas de Blocos de Confiabilidade.


Regras do jogo :  
- O jogador deve iniciar a jornada de perguntas e responder com exatidão. 8 perguntas e 3 delas são desafiadoras, valendo o dobro da pontuação.
- O jogador possui 3 vidas. Ao errar 3 vezes o jogador volta ao início. 
- O jogador não recupera vida, porém acertando a pergunta desafio adiciona +2 ao score
