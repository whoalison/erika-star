Erika Star é um jogo desenvolvido como projeto da disciplina de Modelagem e Simulação de Sistemas Computacionais. O jogo acompanha a jornada de Erika sobre o aprendizado de Diagrama de Blocos de Confiabilidade.

Regras do jogo :

O jogador deve iniciar a jornada de perguntas e responder com exatidão. 
Ao anoitecer as perguntas valem o dobro da pontuação.
O jogador possui 3 vidas. Ao errar 3 vezes o jogador volta ao início.
O jogador não recupera vida, porém acertando a pergunta desafio adiciona +2 ao score.
